## ✅ Чек-лист для тестирования

### 1. Тест приветствия (/start)

**Действие**: Отправьте `/start`

**Ожидаемый результат**:
```
👋 Привет! Я Тамара — бухгалтер.

Помогаю бизнесу навести порядок в финансах, 
рассчитать налоги и сэкономить время.

📊 Хочу показать, как работает бот для 
автоматизации бухгалтерии.

💡 Хотите получать приглашения на встречи 
по средам?
```

✅ Проверьте:
- [ ] Имя "Тамара — бухгалтер" присутствует
- [ ] Текст маркетинговый и привлекательный
- [ ] Есть кнопки "Да" и "Нет"
- [ ] НЕТ упоминаний "Татьяна" или "Горюнова"

---

### 2. Тест главного меню

**Действие**: Нажмите "Да" или отправьте `/menu`

**Ожидаемое меню**:
```
┌─────────────────────────────────┐
│ 🔔 Мои встречи │ 📅 Ближайшие   │
├─────────────────────────────────┤
│ 📝 Записаться  │ 🧩 Админ-панель│
│                │    (демо)      │
├─────────────────────────────────┤
│         🚫 Отписаться           │
└─────────────────────────────────┘
```

✅ Проверьте:
- [ ] Есть кнопка "🧩 Админ-панель (демо)"
- [ ] НЕТ кнопки "✨ О Татьяне"
- [ ] Все остальные кнопки на месте

---

### 3. Тест демо-админки (через меню)

**Действие**: Нажмите "🧩 Админ-панель (демо)"

**Ожидаемое сообщение**:
```
⚙️ Это демо-версия админ-панели

🧪 В этом шоуруме функции доступны всем 
пользователям для демонстрации возможностей.

🔐 В реальном боте доступ к админ-панели 
ограничен и доступен только администраторам.

📊 Доступные функции:
• Просмотр статистики
• Список зарегистрированных
• Регистрации на встречи
• Экспорт данных в CSV

💡 Используйте команду /admin для доступа 
к полной панели управления.
```

✅ Проверьте:
- [ ] Объясняется, что это демо
- [ ] Есть кнопка "📊 Открыть админ-панель"
- [ ] Есть кнопка "🔙 Назад в меню"

**Действие 2**: Нажмите "📊 Открыть админ-панель"

✅ Проверьте:
- [ ] Открылась полная админ-панель
- [ ] Есть уведомление "🧪 DEMO MODE"

---

### 4. Тест команды /admin

**Действие**: Отправьте `/admin`

**Ожидаемое сообщение**:
```
🧪 DEMO MODE - Админ-панель доступна всем

👨‍💼 Админ-панель

Выберите действие:
```

**Кнопки**:
- 👥 Список зарегистрированных
- 📊 Общая статистика
- 📂 Экспорт базы
- 📋 Регистрации на встречи

✅ Проверьте:
- [ ] Админка открывается без ограничений
- [ ] Показывается "🧪 DEMO MODE"
- [ ] Все 4 кнопки присутствуют

**Действие 2**: Нажмите каждую кнопку

✅ Проверьте:
- [ ] "Список зарегистрированных" - показывает пользователей
- [ ] "Общая статистика" - показывает цифры
- [ ] "Регистрации на встречи" - показывает список встреч
- [ ] "Экспорт базы" - генерирует и отправляет CSV файлы

---

### 5. Тест других функций

#### 5.1 Мои встречи
**Действие**: Нажмите "🔔 Мои встречи"

✅ Проверьте:
- [ ] Показывает список ваших регистраций
- [ ] Или сообщает, что регистраций нет

#### 5.2 Ближайшие встречи
**Действие**: Нажмите "📅 Ближайшие встречи"

✅ Проверьте:
- [ ] Показывает расписание будущих встреч
- [ ] Указаны даты, темы, время

#### 5.3 Записаться на встречу
**Действие**: Нажмите "📝 Записаться на встречу"

✅ Проверьте:
- [ ] Показывает список доступных встреч
- [ ] Можно выбрать и записаться
- [ ] После записи приходит подтверждение

## 📊 Быстрая проверка

Минимальный тест (2 минуты):

1. `/start` → проверить приветствие от "Тамара — бухгалтер"
2. Нажать "Да" → проверить наличие кнопки "🧩 Админ-панель (демо)"
3. Нажать кнопку → проверить объяснение демо
4. "Открыть админ-панель" → проверить "🧪 DEMO MODE"
5. `/admin` → проверить прямой доступ

**Время**: ~2 минуты
**Критичность**: 🔴 Высокая
